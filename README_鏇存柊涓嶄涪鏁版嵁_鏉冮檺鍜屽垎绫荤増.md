# 学生状态记录 v3：更新不丢数据、权限显示、被动违规归因

## 先看结论

这版不生成 APK，需要你用 GitHub Actions 打包。

重点改动：

1. 去掉计划功能，只保留记录、统计、反思。
2. 状态可以自己添加、删除。
3. 状态分类可以自己添加、删除。
4. App/违规类型可以自己添加、删除。
5. 学习违规不能主动选择，只能在学习状态下切出 App 后被动生成。
6. 有权限显示模块，缺少“使用情况访问权限”时会提示。
7. 打包方式改成固定签名，后续覆盖安装不需要卸载，避免数据丢失。

---

## 不丢数据的核心规则

Android App 数据能不能保留，主要看三点：

1. applicationId 不能变：本项目固定为 `com.yanjia.studentreflect`
2. 签名不能变：本项目加入固定签名文件 `app/keystore/student-reflect-update.jks`
3. versionCode 要变大：本版为 `versionCode 11`

以后你只要不改这三样，手机上直接安装新版 APK 覆盖旧版，数据会保留。

注意：如果你当前手机里安装的旧版 APK 是 GitHub 默认临时 debug 签名，第一次换到这个固定签名版时，可能会提示“签名不一致/应用未安装”。如果出现这个提示，不是源码错，而是旧版签名和新版固定签名不同。这个问题 Android 系统不能绕过。

从本版开始，后续每次更新都可以保留数据。

## GitHub 更新步骤

1. 解压这个源码包。
2. 打开解压后的文件夹。
3. 进入里面，不要上传最外层空壳文件夹。
4. 选中这些内容上传到你原来的 GitHub 仓库：

```
.github
app
build.gradle
settings.gradle
README_更新不丢数据_权限和分类版.md
```

5. 提交 Commit changes。
6. 打开 Actions。
7. 运行 `Build Android APK Stable Update`。
8. 下载 Artifacts 里的 `student-reflect-v3-stable-update-apk`。
9. 解压得到 `app-debug.apk`。
10. 直接安装覆盖旧版，不要卸载。

## 如果覆盖安装失败

如果手机提示：

- 应用未安装
- 签名不一致
- 与已安装应用签名冲突

说明旧版 APK 和新版 APK 签名不同。

这种情况下不要急着卸载。如果旧版里已有重要数据，先在旧版里尝试导出数据。新版提供“导出全部数据/导入全部数据”。

如果旧版按钮完全不能用，数据又很重要，就先别卸载，把报错截图发给我，再决定怎么迁移。

## 使用情况访问权限

为了自动判断学习时切出去用了哪个 App，需要打开权限：

```
设置
→ 应用
→ 特殊应用权限
→ 使用情况访问权限
→ 学生状态记录
→ 允许
```

OPPO 如果找不到，就在设置搜索：

```
使用情况访问权限
```

没有这个权限时，App 仍然可以连续计时，但不能自动读取手机使用时间。你可以先用“粘贴导入手机使用记录”。

## 被动违规逻辑

学习状态下：

```
英语学习 / 数学学习 / 深度学习
```

如果你切出本 App，返回后系统会自动把这段记录为：

```
自动记录：学习违规离开
```

你不能手动选择这个状态。

然后系统会读取手机使用时间，找出这段时间里重叠最多的 App：

```
违规时间：20:10-20:22
手机记录：20:08-20:20 抖音
手机记录：20:20-20:23 微信
```

重叠最多是抖音，所以自动判定：

```
违规 App：抖音
违规类型：短视频/娱乐刷屏
```

如果这个 App 还没有归类，会出现在“未分类 App”里。你只需要归类一次，以后自动处理。

## 分数系统

综合分 = 六项加权：

```
学习投入：30%
深度学习：15%
专注纪律：20%
娱乐控制：15%
生活状态：10%
反思复盘：10%
```

重点：

- 违规次数越多，专注纪律越低。
- 违规时间越长，专注纪律越低。
- 违规类型严重度越高，扣分越重。
- 游戏/娱乐时间过高，会扣娱乐控制分。
- 深度学习时间不足，会影响深度学习分。
- 反思没写，会影响反思复盘分。

## 不要随便改的地方

不要改：

```
applicationId 'com.yanjia.studentreflect'
versionCode 11 只允许以后改大，不要改小
app/keystore/student-reflect-update.jks 不要删
```

以后更新时，versionCode 可以改成 12、13、14……
